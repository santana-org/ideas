# 🚀 Goodra
A CLI tool that scans development environments, detects large directories and common dev artifacts like `node_modules` and `build outputs`, provides `dry-run previews`, and summarizes recoverable disk space before cleanup.
## 🧠 Problem

**This project solves:**
- Existing disk usage tools report large files but lack context about development artifacts like dependency directories, build outputs, and language-specific caches.

## 💡 Features

**List the main capabilities:**
- 📦 Scan workspaces by size.
- 🧪 Safe dry-run mode.
- 🧹 Selective cleanup.

## 🧩 Advanced Features
- ⚙️ Configurable rules.
- ⚠️ Risk classification.
